# modal-modal-modal

Let's investigate complex biological questions using large, multimodal datasets in the public domain. 

## Background

You're seeing it everywhere: multimodal. What does it mean? Why has it become the prevailing direction of biological reserach across academic, biotech, and pharma sectors? Are there canonical analyses or methodologies that leverage these diverse datasets? What are teams learning via this approach? 

This project will investigate these questions directly using data from public resources so anyone can reproduce findings herein.
